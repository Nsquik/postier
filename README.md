# Postier

### Website that works with https://gorest.co.in/ API. 

# Stack
## Front-End

- React

- React hooks API

- Redux

- Ant Design

- Typescript

- Formik



## Run locally

#### Clone repo
#### Create .env file in root dir ( NOT src )
###### Inside .env

```
REACT_APP_API_KEY=your GOREST.CO.IN api KEY

```
